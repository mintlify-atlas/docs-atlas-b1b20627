title: Using the explorer

---

ETHGlobal Explorer provides a powerful web interface for browsing and discovering projects from ETHGlobal hackathons dating back to 2020.

## Accessing the explorer

<Steps>
  <Step title="Navigate to the website">
    Visit [ethglobalexplorer.com](https://www.ethglobalexplorer.com) in your web browser to access the explorer.
  </Step>
  
  <Step title="Browse projects">
    The main page displays a table of hackathon projects with 100 projects per page. Projects are shown with their name, description, event, and prizes won.
  </Step>
</Steps>

## Understanding the interface

The explorer interface consists of several key components:

### Search and filter controls

At the top of the page, you'll find a row of controls to help you find specific projects:

- **Search bar**: Text input for searching projects by title
- **Events dropdown**: Filter projects by specific hackathon events
- **Prizes dropdown**: Filter projects by prize sponsors
- **Tags dropdown**: Filter projects by technology categories

<Tip>
You can use multiple filters together to narrow down your search. For example, combine an event filter with a prize filter to see all projects that won a specific prize at a particular hackathon.
</Tip>

### Projects table

The main table displays project information in four columns:

- **Name**: Clickable project title that links to the full project page on ETHGlobal
- **Description**: Brief description of what the project does
- **Event**: The hackathon where the project was submitted (displayed in short format like "Bangkok '24" or "NYC '25")
- **Prizes**: Visual badges showing prize sponsor logos for all prizes the project won

<Info>
The explorer contains projects from HackMoney 2020 onwards. Prize data (sponsor logos) is available from HackFS 2021 and later events.
</Info>

## Viewing project details

To learn more about a specific project:

<Steps>
  <Step title="Click the project name">
    Project names in the table are blue hyperlinks. Click any project name to open the full project page.
  </Step>
  
  <Step title="View on ETHGlobal">
    The link opens the project's showcase page on ETHGlobal's website in a new tab, where you can see the full description, team members, demo video, and code repository.
  </Step>
</Steps>

## Browsing events

ETHGlobal Explorer also includes a dedicated events page that lists all ETHGlobal hackathons:

<Steps>
  <Step title="Navigate to the events page">
    Access the events page at [ethglobalexplorer.com/events](https://www.ethglobalexplorer.com/events) to view a comprehensive list of all hackathons.
  </Step>
  
  <Step title="View event information">
    The events table displays event names and placeholder fields for dates, locations, participants, and project counts (currently being populated).
  </Step>
</Steps>

<Note>
The events page is currently under development. Additional event metadata including dates, locations, and participant statistics will be added in future updates.
</Note>

## Navigating pages

The explorer displays 100 projects per page. Use the pagination controls at the bottom of the page:

- **Previous button**: Navigate to the previous page of results
- **Page indicator**: Shows current page number and total pages (e.g., "Page 1 of 45")
- **Next button**: Navigate to the next page of results

<Note>
Pagination buttons are automatically disabled when you're at the first or last page, or while the page is loading new results.
</Note>

## Data coverage

The explorer includes:

- **Projects**: All projects from HackMoney 2020 to the most recent ETHGlobal events
- **Events**: 65+ hackathon events including ETHGlobal flagship events, online hackathons, and themed events
- **Prizes**: 250+ prize sponsors and partners
- **Tags**: Technology categories including AI Agents, Stablecoins, NFTs, and DAOs

<Info>
Some older hackathons (pre-2020) were hosted on other platforms like Devpost and are not included in this explorer.
</Info>
