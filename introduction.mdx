# ETHGlobal Explorer

ETHGlobal Explorer is a comprehensive, searchable database of over 13,000 hackathon projects from ETHGlobal events spanning from 2020 to present. Discover innovative blockchain projects, filter by events, prizes, and tags, and explore the cutting edge of Ethereum development.

## What is ETHGlobal Explorer?

ETHGlobal Explorer provides a powerful interface to search and discover hackathon projects from 60+ ETHGlobal events. Whether you're looking for inspiration, researching specific technologies, or exploring prize-winning projects, ETHGlobal Explorer makes it easy to navigate thousands of projects with advanced filtering capabilities.

## Database coverage

ETHGlobal Explorer includes comprehensive data from ETHGlobal's showcase page:

- **Events**: From HackMoney 2020 to the latest hackathons including Bangkok, San Francisco, Singapore, and online events
- **Prize data**: Prize logos and sponsor information available from HackFS 2021 onwards
- **Project details**: Title, description, event information, and prize winners for each project

<Note>
  Some historical ETHGlobal events (pre-2020) were hosted on platforms like Devpost and are not included in the database. Projects from ETHNYC and earlier events are not available on ETHGlobal's showcase page.
</Note>

## Getting started

Start exploring projects by:

1. **Searching by keyword** - Find projects by name or technology
2. **Filtering by event** - Browse projects from specific hackathons
3. **Filtering by prize** - Discover projects that won specific sponsor prizes
4. **Filtering by tag** - Explore projects by category (AI Agents, Stablecoins, NFTs, DAOs)

Visit [ethglobalexplorer.com](https://ethglobalexplorer.com) to start exploring.
